# Small-Company-Network-Design
Enterprise-style network infrastructure designed in Cisco Packet Tracer with VLAN isolation, inter-VLAN routing, ACLs, DHCP, DNS, NAT, and secure connectivity across multiple departments.
**
**Small Company Network Design – Cisco Packet Tracer****

Overview

This project simulates a small company network infrastructure using Cisco Packet Tracer. The network is designed to provide secure communication, efficient traffic management, and centralized network services across multiple departments.

The implementation includes VLAN segmentation, inter-VLAN routing, DHCP, DNS, NAT, ACLs, and network monitoring to reflect real-world enterprise networking practices.

Objectives
Segment network traffic using VLANs.
Enable communication between departments through inter-VLAN routing.
Improve security using Access Control Lists (ACLs).
Automate IP address assignment with DHCP.
Provide centralized DNS services.
Configure NAT for internet connectivity.
Implement basic network monitoring and management.
Network Architecture
HR Department (VLAN 10)
Dedicated VLAN for HR employees.
DHCP scope configured for automatic IP assignment.
ACLs applied to protect sensitive employee and payroll data.
End-user connectivity support and testing.
Sales Department (VLAN 20)
Separate VLAN for Sales operations.
Inter-VLAN routing enabled for controlled access to shared resources.
NAT configured for internet access.
Network access troubleshooting and validation.
IT Department (VLAN 30)
Dedicated VLAN for IT administrators.
Administrative access to network devices and servers.
Centralized DNS and DHCP services.
STP configured for loop prevention.
Network monitoring implemented for operational visibility.
